Instrucciones de uso

Para usar la aplicacion, debes subir un archivo con formato .obj. Una vez cargado, este se mostrara en el visualizador y podras interactuar con este. Puedes escalar el modelo usando los botones de control.

  • Haz click y desliza el mouse sobre el visualizador para girar la figura
  • Usa la rueda del raton para acercar o alejar el modelo

Opcionalmente, puedes crear una cuenta y guardas modelos para poder visualizarlos posteriormente. Los modelos a guardar tienen un limite de 10MB.

Visor3D

Visor3D es una aplicacion web que permite visualizar modelos 3D en el formato .obj, escalar, rotar y cambiar el material del objeto. Tambien puedes crear tu propia cuenta y guardas modelos 3D para visualizar donde quieras y cuando quieras.
